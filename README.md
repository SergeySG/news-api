
## Description
News Api using [GNews API](https://gnews.io/)



## Documentation

You can find Swagger Documentation using http://localhost:3000/api/doc

## Installation

```bash
$ npm install
```

Add GNEws api key to `.env`

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
